# gpod
POD (precise orbit determination) of GNSS satellites

# 简介
支持GPS/BDS/GLONASS/GALILEO精密定轨，精度厘米级

测试数据在百度云盘（125个全球IGS站）：链接: https://pan.baidu.com/s/15ZHs17U9rzWYmUXBbEvZwQ 提取码: ywaw
